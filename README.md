## Démarche

Ces articles refusent deux caricatures symétriques : la Belgique modèle de compromis démocratique, et la Belgique État dysfonctionnel par essence. Ils partent d'une question précise : quand des institutions documentées par les plus hautes instances internationales comme structurellement défaillantes, pourquoi ces défaillances persistent-elles ?

La réponse n'est ni la corruption individuelle, ni la malveillance. Elle est institutionnelle — ce que la doctrine appelle la **particratie** : un système où les acteurs qui devraient réformer sont précisément ceux qui bénéficient du statu quo.

**Toutes les affirmations sont fondées sur des sources primaires.** Les références complètes figurent à la fin de chaque article.
